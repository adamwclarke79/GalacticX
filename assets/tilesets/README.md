# GalacticX Tilesets

Place PixelLab-generated and imported low top-down tilesets here.

Expected convention:

- `assets/tilesets/<tileset_name>/`
- Include source PNGs, generated metadata, and Godot `TileSet` resources when available.
- Tile collision should be authored in the Godot `TileSet` whenever practical.

Current imports:

- `spaceship_tileset/`: extracted spaceship interior tile/object sheets from `Spaceship tileset.rar`.
